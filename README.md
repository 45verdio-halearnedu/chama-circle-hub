# Chama Circle Hub

> **Empowering African communities to save, lend, invest, and grow — together.**

---

## 🔍 Project Overview

**Chama Circle Hub** is a powerful community-driven digital finance platform built to modernize how savings groups (commonly known as *chamas* in Africa) operate, manage funds, access loans, and grow wealth collaboratively. It's designed for both individual users and organized groups — with integrated features for **admins**, **banks**, and **investors**.

---

## 🌟 Key Features

### 🢑 Chama Group System

* Create or join a chama
* Role assignments (Chairperson, Treasurer, Secretary)
* Group chat, voting, and member management
* Contribution rules, savings rounds, and loan policies
* Performance tracking & reports

### 💼 Loan & Lending Management

* Group and personal loan applications
* Loan status tracking and history
* Peer-based lending and voting approval
* Collateral and interest settings
* Admin & investor dashboard

### 📊 Analytics & Reporting

* Group savings performance
* Member contributions and repayments
* Downloadable statements
* Reputation scores

### 🔐 Authentication & Roles

* Email, phone number, and password-based login
* Role-based dashboards: Member, Admin, Banker
* Identity verification via national ID or passport

### 🏦 Bank & Admin Portals

* Bank API dashboard (loan marketplace, compliance logs)
* Admin analytics, loan oversight, messaging center
* User management and group audits
* Chama performance monitoring

### 📱 Wallets & Payments

* Group and individual wallets
* Integration with **M-Pesa**, Airtel Money, Banks
* QR code payments, deposits, and withdrawals
* Payment reminders and scheduling

### 📉 Investments & Staking

* Investment opportunities for members
* Staking of group savings
* Yield tracking and reward distribution

### 🎓 Community Education

* Financial literacy modules
* Group challenges and incentives
* Voting systems for decision-making

## 🚜 Tech Stack

| Layer     | Technology                                                    |
| --------- | ------------------------------------------------------------- |
| Frontend  | React + TypeScript + Tailwind CSS + shadcn/ui                 |
| Backend   | [Supabase](https://supabase.com) (PostgreSQL, Auth, Realtime) |
| Payments  | M-Pesa, Airtel, Bank Integrations                             |

## 💡 Vision

Chama Circle Hub envisions a **decentralized future of African community finance** where **transparency, access, and collaboration** empower groups and individuals to build sustainable financial ecosystems.
